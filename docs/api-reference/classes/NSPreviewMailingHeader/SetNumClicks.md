# SetNumClicks(Integer numClicks)

Number of recipients that clicked at least 1 link in the mailing.

**Parameter:** 
 - **numClicks** Integer

```crmscript
NSPreviewMailingHeader thing;
Integer numClicks;
thing.SetNumClicks(numClicks);
```

# SetPage1LineNo(Integer page1LineNo)

Line no on View Page 1, used if MDO flags are OFF. 0 = this field is not visible on page 1

**Parameter:** 
 - **page1LineNo** Integer

```crmscript
NSUserDefinedFieldInfo thing;
Integer page1LineNo;
thing.SetPage1LineNo(page1LineNo);
```

Required field when creating ticket in offline mode

**Parameter:** 
 - **offlineFields** NSWidgetRequiredFields
     - Enum: 0 = None 
     - Enum: 1 = Email 
     - Enum: 2 = Name 
     - Enum: 4 = Company 
     - Enum: 8 = Phone 

```crmscript
NSChatWidgetSettings thing;
NSWidgetRequiredFields offlineFields;
thing.SetOfflineFields(offlineFields);
```

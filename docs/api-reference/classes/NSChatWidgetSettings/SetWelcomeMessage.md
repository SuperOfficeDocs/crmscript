# SetWelcomeMessage(String welcomeMessage)

**Parameter:** 
 - **welcomeMessage** String

```crmscript
NSChatWidgetSettings thing;
String welcomeMessage;
thing.SetWelcomeMessage(welcomeMessage);
```

# SetDisplayText(String displayText)

A short description, suitable for display in a popup menu or dropdown list of versions.

**Parameter:** 
 - **displayText** String

```crmscript
NSVersionInfo thing;
String displayText;
thing.SetDisplayText(displayText);
```

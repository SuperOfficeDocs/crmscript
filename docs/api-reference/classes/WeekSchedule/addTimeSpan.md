# WeekSchedule.addTimeSpan(DateTime start, TimeSpan add)

Adds a timespan to the schedule.

## Parameters

 - start: Start time for calculation
 - add: The timespan to add (2 days, 1 hours, 5 minutes, 30 seconds...)

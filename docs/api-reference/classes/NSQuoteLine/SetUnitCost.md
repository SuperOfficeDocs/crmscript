# SetUnitCost(Float unitCost)

The cost price per unit for this product. May be filled in by connector if it has the Provide-Cost capability.

**Parameter:** 
 - **unitCost** Float

```crmscript
NSQuoteLine thing;
Float unitCost;
thing.SetUnitCost(unitCost);
```

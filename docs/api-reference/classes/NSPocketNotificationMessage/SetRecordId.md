# SetRecordId(Integer recordId)

If the message is about some specific database record, this could be its primary key

**Parameter:** 
 - **recordId** Integer

```crmscript
NSPocketNotificationMessage thing;
Integer recordId;
thing.SetRecordId(recordId);
```

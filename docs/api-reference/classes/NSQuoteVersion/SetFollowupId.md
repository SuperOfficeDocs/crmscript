# SetFollowupId(Integer followupId)

Link to a follow-up activity, created when this quote version was sent to the customer.

**Parameter:** 
 - **followupId** Integer

```crmscript
NSQuoteVersion thing;
Integer followupId;
thing.SetFollowupId(followupId);
```

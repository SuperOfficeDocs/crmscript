# SetTooltip(String tooltip)

Set the tooltip description for the DeliveryType list item.

**Parameter:** 
 - **tooltip** String DeliveryType tooltip description.

```crmscript
NSDeliveryType thing;
String descr = "something new";
thing.SetTooltip(descr);
```

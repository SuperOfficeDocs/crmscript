# NSPreviewMailing GetPreviewMailing(Integer shipmentAddrId)

**Parameters:**

**Returns:** NSPreviewMailing

```crmscript

Integer shipmentAddrId;
NSPreviewMailing res = agent.GetPreviewMailing(shipmentAddrId);
```

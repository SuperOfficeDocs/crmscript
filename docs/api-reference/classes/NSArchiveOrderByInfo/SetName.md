# SetName(String name)

Programmatic name of the column to order by, semantically the same as the NSArchiveColumnInfo.Name.

**Parameter:** 
 - **name** String

```crmscript
NSArchiveOrderByInfo thing;
String name;
thing.SetName(name);
```

# sleep(Float seconds)

Sleep (give up CPU) for some time.
Useful in some CPU intensive loops to maintain a responsive system.

## Example

```crmscript
sleep(01); //sleeps for about one-tenth of a second.
```

* **seconds:** Number of seconds to sleep.

# SetType(Integer type)

How is the return value to be understood by the caller

**Parameter:** 
 - **type** Integer
     - Enum: 0 = None 
     - Enum: 1 = Message 
     - Enum: 2 = SoProtocol 
     - Enum: 3 = CustomGui 
     - Enum: 4 = Other 
     - Enum: 5 = URL 

```crmscript
NSReturnInfo thing;
Integer type;
thing.SetType(type);
```

# SetSilent(Bool silent)

Indicates wether or not the message will vibrate/play a sound when delivered

**Parameter:** 
 - **silent** Bool

```crmscript
NSPocketNotificationMessage thing;
Bool silent;
thing.SetSilent(silent);
```

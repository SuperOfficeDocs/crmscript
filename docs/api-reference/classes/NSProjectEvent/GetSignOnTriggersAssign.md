# Bool GetSignOnTriggersAssign()

If true, the sign on task should be created as an Assigned task, triggering the invitation dialog

**Returns:** Bool

```crmscript
NSProjectEvent thing;
Bool signOnTriggersAssign  = thing.GetSignOnTriggersAssign();
```

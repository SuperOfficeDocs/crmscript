# SetTooltip(String tooltip)

Set the tooltip description for the NSReason list item.

**Parameter:** 
 - **tooltip** String NSReason tooltip description.

```crmscript
NSReason thing;
String descr = "something new";
thing.SetTooltip(descr);
```

# SetIsDefaultNow(Bool isDefaultNow)

Should the current date + time be the default?

**Parameter:** 
 - **isDefaultNow** Bool

```crmscript
NSFieldInfoDateTime thing;
Bool isDefaultNow;
thing.SetIsDefaultNow(isDefaultNow);
```

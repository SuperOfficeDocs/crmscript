# SetDataRights(DataRights dataRights)

Data rights matrix - defines role's access to data owned by current user, users in same group, and other users.

**Parameter:** 
 - **dataRights** DataRights

```crmscript
NSRoleEntity thing;
DataRights dataRights;
thing.SetDataRights(dataRights);
```

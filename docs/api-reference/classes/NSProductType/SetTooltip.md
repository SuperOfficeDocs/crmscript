# SetTooltip(String tooltip)

Set the tooltip description for the NSProductType list item.

**Parameter:** 
 - **tooltip** String NSProductType tooltip description.

```crmscript
NSProductType thing;
String descr = "something new";
thing.SetTooltip(descr);
```

# SetMinEarningPercent(Integer minEarningPercent)

**Parameter:** 
 - **minEarningPercent** Integer

```crmscript
NSSaleTypeEntity thing;
Integer minEarningPercent;
thing.SetMinEarningPercent(minEarningPercent);
```

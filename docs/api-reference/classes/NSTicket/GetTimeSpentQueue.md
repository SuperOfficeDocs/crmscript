# Integer GetTimeSpentQueue()

The total time (seconds) within the priority's office hours the ticket has been in a queue status, not including current state

**Returns:** Integer

```crmscript
NSTicket thing;
Integer timeSpentQueue  = thing.GetTimeSpentQueue();
```

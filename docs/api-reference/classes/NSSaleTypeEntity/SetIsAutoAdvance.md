# SetIsAutoAdvance(Bool isAutoAdvance)

Does the sale stage advance automatically, when the last guided activity in a stage is completed?

**Parameter:** 
 - **isAutoAdvance** Bool

```crmscript
NSSaleTypeEntity thing;
Bool isAutoAdvance;
thing.SetIsAutoAdvance(isAutoAdvance);
```

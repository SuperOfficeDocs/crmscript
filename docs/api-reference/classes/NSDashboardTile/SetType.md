# SetType(Integer type)

The tile type, what kind of tile this is

**Parameter:** 
 - **type** Integer
     - Enum: 0 = None 
     - Enum: 1 = Chart 
     - Enum: 2 = Web 
     - Enum: 3 = List 
     - Enum: 4 = Bignum 

```crmscript
NSDashboardTile thing;
Integer type;
thing.SetType(type);
```

# String GetCategoryFullname()

The fullname (compiled from parents&apos; names) for this category. E.g. &apos;norway/product A/support&apos;.

**Returns:** String

```crmscript
NSTicket thing;
String categoryFullname  = thing.GetCategoryFullname();
```

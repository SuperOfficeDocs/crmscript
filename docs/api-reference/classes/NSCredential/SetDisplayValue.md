# SetDisplayValue(String displayValue)

The value displayed to the user. this will typically be the users login name in active directory.

**Parameter:** 
 - **displayValue** String

```crmscript
NSCredential thing;
String displayValue;
thing.SetDisplayValue(displayValue);
```

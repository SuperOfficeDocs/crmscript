# Void DeleteAllHeadingsFromListDefinition(Integer id)

Delete all headings for list resolved by the provided id.

**Parameters:**
 - **id** The id of the list to look up.

```crmscript
NSListAgent agent;
Integer id;
agent.DeleteAllHeadingsFromListDefinition(id);
```

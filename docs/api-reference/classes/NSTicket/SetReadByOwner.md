# SetReadByOwner(DateTime readByOwner)

The datetime for when the ticket last was read by the owner.

**Parameter:** 
 - **readByOwner** DateTime

```crmscript
NSTicket thing;
DateTime readByOwner;
thing.SetReadByOwner(readByOwner);
```

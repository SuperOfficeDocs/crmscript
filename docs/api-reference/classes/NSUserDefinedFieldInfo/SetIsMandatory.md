# SetIsMandatory(Bool isMandatory)

0 = no, 1 = yes (field must be filled out)

**Parameter:** 
 - **isMandatory** Bool

```crmscript
NSUserDefinedFieldInfo thing;
Bool isMandatory;
thing.SetIsMandatory(isMandatory);
```

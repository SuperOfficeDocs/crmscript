# ReplyTemplate.getHtmlBody(Integer language)

Returns the Reply template html body, in the language given by language

## Parameters

 - language as an integer, see below

## Languages

Norwegian = 0,
English = 1,
German = 2,
Swedish = 3,
Danish = 4,
Dutch = 5,

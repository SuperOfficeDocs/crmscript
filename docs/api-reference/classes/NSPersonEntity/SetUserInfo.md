Information about the user if this person is a user.  If IsAssociate (e.g. is user is true) the NSUserInfo will be provided.

**Parameter:** 
 - **userInfo** NSUserInfo

```crmscript
NSPersonEntity thing;
NSUserInfo userInfo;
thing.SetUserInfo(userInfo);
```

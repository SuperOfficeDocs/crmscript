# SetIsSubscription(Bool isSubscription)

Is this a subscription product, sold in repeating intervals/amounts?

**Parameter:** 
 - **isSubscription** Bool

```crmscript
NSProduct thing;
Bool isSubscription;
thing.SetIsSubscription(isSubscription);
```

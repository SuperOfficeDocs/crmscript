Change password for a user.

**Parameters:**
 - **associateName** Username to set password for
 - **password** New password

**Returns:** Bool

```crmscript
NSUserAgent agent;
String associateName;
String password;
Bool res = agent.SetPasswordFromName(associateName, password);
```

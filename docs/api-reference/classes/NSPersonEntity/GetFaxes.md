# NSEntityElement[] GetFaxes()

Returns a collection of fax numbers that belong to the contact person.

**Returns:** NSEntityElement[]

```crmscript
NSPersonEntity thing;
NSEntityElement[] faxes  = thing.GetFaxes();
```

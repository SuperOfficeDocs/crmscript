# Integer GetMultiWordOperator()

Starts with, contains or exact match

**Returns:** Integer

     - Enum: 1 = Contains 
     - Enum: 2 = StartsWith 
     - Enum: 3 = ExactMatch 

```crmscript
NSFreeText thing;
Integer multiWordOperator  = thing.GetMultiWordOperator();
```

# SetHasImage(Bool hasImage)

True if the project has an image. (This is the image that is displayed in the CRM client)

**Parameter:** 
 - **hasImage** Bool

```crmscript
NSProjectEntity thing;
Bool hasImage;
thing.SetHasImage(hasImage);
```

# SetSatStart(DateTime satStart)

The work hour start for Saturdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **satStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime satStart;
thing.SetSatStart(satStart);
```

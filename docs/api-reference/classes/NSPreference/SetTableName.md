# SetTableName(String tableName)

For DisplayType=PrefDescValueType.ListTableRef, TableName will contain the name of the table which the value reference to.

**Parameter:** 
 - **tableName** String

```crmscript
NSPreference thing;
String tableName;
thing.SetTableName(tableName);
```

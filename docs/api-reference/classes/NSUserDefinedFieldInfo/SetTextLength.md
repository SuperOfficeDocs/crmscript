# SetTextLength(Integer textLength)

Length (in characters) of a text field, 0 for other types

**Parameter:** 
 - **textLength** Integer

```crmscript
NSUserDefinedFieldInfo thing;
Integer textLength;
thing.SetTextLength(textLength);
```

# String[] GetCustomTags()

Array with custom tags and values. Array may be of unlimited height, but need to be exactly 2 in width.

**Returns:** String[]

```crmscript
NSTemplateVariablesParameters thing;
String[] customTags  = thing.GetCustomTags();
```

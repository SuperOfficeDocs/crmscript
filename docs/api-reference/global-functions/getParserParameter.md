# String getParserParameter()

Returns the value of the given parameter in the global Parser instance. You can optionally pass the zero-based index of the value.

## Functions

* String getParserParameter(String name)
* String getParserParameter(String name, Integer index)

## Parameters

| Parameter | Type | Description |
|---|---|---|
| name | String | The name of the parameter |
| index | Integer | Optional. The zero-based index of the value to return |

# Integer GetPage1LineNo()

Line no on View Page 1, used if MDO flags are OFF. 0 = this field is not visible on page 1

**Returns:** Integer

```crmscript
NSUserDefinedFieldInfo thing;
Integer page1LineNo  = thing.GetPage1LineNo();
```

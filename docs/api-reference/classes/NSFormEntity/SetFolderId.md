# SetFolderId(Integer folderId)

The folder which this form belongs to. -1 indicates that the shipment is on the root

**Parameter:** 
 - **folderId** Integer

```crmscript
NSFormEntity thing;
Integer folderId;
thing.SetFolderId(folderId);
```

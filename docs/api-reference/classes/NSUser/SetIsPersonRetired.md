# SetIsPersonRetired(Bool isPersonRetired)

True if the user is retired and should have no rights, not appear in lists, etc.

**Parameter:** 
 - **isPersonRetired** Bool

```crmscript
NSUser thing;
Bool isPersonRetired;
thing.SetIsPersonRetired(isPersonRetired);
```

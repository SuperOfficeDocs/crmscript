# SetFieldWidth(Integer fieldWidth)

field width in pixels, 0 = 'enough' (from font)

**Parameter:** 
 - **fieldWidth** Integer

```crmscript
NSUserDefinedFieldInfo thing;
Integer fieldWidth;
thing.SetFieldWidth(fieldWidth);
```

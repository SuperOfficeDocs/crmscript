# SetDisplayValue(String displayValue)

Display value, populated when asked for. Suitable for binding display-datasource in lists, etc. Formatted using CultureDataFormatter.

**Parameter:** 
 - **displayValue** String

```crmscript
NSPreference thing;
String displayValue;
thing.SetDisplayValue(displayValue);
```

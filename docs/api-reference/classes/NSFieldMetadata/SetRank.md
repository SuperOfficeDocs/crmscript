# SetRank(Integer rank)

A way to set the order of the fields. Lowest value will be displayed first/over the fields with other values.

**Parameter:** 
 - **rank** Integer

```crmscript
NSFieldMetadata thing;
Integer rank;
thing.SetRank(rank);
```

# Integer GetCompleted()

Document Completed state. This is the part of the Status property. 

**Returns:** Integer

     - Enum: 0 = Unknown 
     - Enum: 1 = NotStarted 
     - Enum: 2 = Started 
     - Enum: 3 = Completed 

```crmscript
NSDocumentEntity thing;
Integer completed  = thing.GetCompleted();
```

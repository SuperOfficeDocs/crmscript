# String GetNetServerSmsProvider()

Name of the selected provider, i.e. CM, Compaya SMS (cpsms.dk), Intelecom SMS, PSWinCom SMS, SMS Teknik or TXTLocal.

**Returns:** String

```crmscript
NSSmsConfig thing;
String netServerSmsProvider  = thing.GetNetServerSmsProvider();
```

# SetTooltip(String tooltip)

Set the tooltip description for the NSCompetitor list item.

**Parameter:** 
 - **tooltip** String NSCompetitor tooltip description.

```crmscript
NSCompetitor thing;
String descr = "something new";
thing.SetTooltip(descr);
```

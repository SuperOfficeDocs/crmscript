# SetSatStop(DateTime satStop)

The work hour stop for Saturdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **satStop** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime satStop;
thing.SetSatStop(satStop);
```

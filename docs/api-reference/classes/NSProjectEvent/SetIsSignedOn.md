# SetIsSignedOn(Bool isSignedOn)

True if the current user is signed on to the event (i.e. is project member)

**Parameter:** 
 - **isSignedOn** Bool

```crmscript
NSProjectEvent thing;
Bool isSignedOn;
thing.SetIsSignedOn(isSignedOn);
```

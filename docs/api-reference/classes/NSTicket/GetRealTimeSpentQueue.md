# Integer GetRealTimeSpentQueue()

The total time (seconds) within 24x7 hours the ticket has been in a queue status, not including current state

**Returns:** Integer

```crmscript
NSTicket thing;
Integer realTimeSpentQueue  = thing.GetRealTimeSpentQueue();
```

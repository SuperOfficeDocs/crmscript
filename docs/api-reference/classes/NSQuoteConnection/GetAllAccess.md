# Bool GetAllAccess()

Is this connection accessible to everyone?  If not, then the QuoteConnectionAccess table tells us who can access it.

**Returns:** Bool

```crmscript
NSQuoteConnection thing;
Bool allAccess  = thing.GetAllAccess();
```

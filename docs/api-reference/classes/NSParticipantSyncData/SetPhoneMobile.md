# SetPhoneMobile(String[] phoneMobile)

Returns a collection of mobile phone numbers that belong to the contact person.

**Parameter:** 
 - **phoneMobile** String[]

```crmscript
NSParticipantSyncData thing;
String[] phoneMobile;
thing.SetPhoneMobile(phoneMobile);
```

# SetSignOffTaskId(Integer signOffTaskId)

If not 0, sign off should cause an Activity of this type to be registered

**Parameter:** 
 - **signOffTaskId** Integer

```crmscript

thing.SetSignOffTaskId(signOffTaskId);
```

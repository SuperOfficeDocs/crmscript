# NSQuoteConnection SaveConnection(NSQuoteConnection connection)

**Parameters:**

**Returns:** NSQuoteConnection

```crmscript
NSQuoteAgent agent;
NSQuoteConnection connection;
NSQuoteConnection res = agent.SaveConnection(connection);
```

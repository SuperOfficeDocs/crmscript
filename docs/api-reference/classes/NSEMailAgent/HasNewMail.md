Fast check for new mail in the database

**Parameters:**
 - **mailItemId** The last known mailitemid
 - **folderId** The folderId for the folder to check for new mail

**Returns:** Bool

```crmscript
NSEMailAgent agent;
Integer mailItemId;
Integer folderId;
Bool res = agent.HasNewMail(mailItemId, folderId);
```

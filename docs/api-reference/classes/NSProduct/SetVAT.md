# SetVAT(Float vAT)

Tax/VAT if available from ERP system. THIS IS A PERCENTAGE. In the Superoffice product register, this field is available in the product admin GUI.

**Parameter:** 
 - **vAT** Float

```crmscript
NSProduct thing;
Float vAT;
thing.SetVAT(vAT);
```

# SetExternalRef(String externalRef)

External reference for document plugin to resolve document identity (Notes ID, e-mail message ID, whatever)

**Parameter:** 
 - **externalRef** String

```crmscript
NSDocumentEntity thing;
String externalRef;
thing.SetExternalRef(externalRef);
```

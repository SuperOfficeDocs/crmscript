# SetThuStart(TimeSpan thuStart)

When opening hours start on Thursdays. Only time-part is used.

**Parameter:** 
 - **thuStart** TimeSpan

```crmscript
NSChatOpeningHours thing;
TimeSpan thuStart;
thing.SetThuStart(thuStart);
```

# SetFriStart(TimeSpan friStart)

When opening hours start on Fridays. Only time-part is used.

**Parameter:** 
 - **friStart** TimeSpan

```crmscript
NSChatOpeningHours thing;
TimeSpan friStart;
thing.SetFriStart(friStart);
```

# Customer.checkTableRights(String tableRight)

Use this function to see if the current user has access to this customer.

tableRight might be:

- select
- update
- insert

## Example

    Customer c;
    
    c.load(3);
    Bool b = c.checkTableRights("select");
    print(b.toString());

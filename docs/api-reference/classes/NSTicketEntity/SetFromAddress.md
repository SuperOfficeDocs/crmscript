# SetFromAddress(String fromAddress)

The from-address used when this ticket got created, e.g. by email

**Parameter:** 
 - **fromAddress** String

```crmscript
NSTicketEntity thing;
String fromAddress;
thing.SetFromAddress(fromAddress);
```

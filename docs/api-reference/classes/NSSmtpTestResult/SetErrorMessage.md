# SetErrorMessage(String errorMessage)

If an error occured, this member will contain a description of the error

**Parameter:** 
 - **errorMessage** String

```crmscript
NSSmtpTestResult thing;
String errorMessage;
thing.SetErrorMessage(errorMessage);
```

# SetWedStop(DateTime wedStop)

The work hour stop for Wednesdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **wedStop** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime wedStop;
thing.SetWedStop(wedStop);
```

# SetFileName(String fileName)

The path to the output file created, if it exists. Will be an empty string if no file was created.

**Parameter:** 
 - **fileName** String

```crmscript
NSExportArchiveResult thing;
String fileName;
thing.SetFileName(fileName);
```

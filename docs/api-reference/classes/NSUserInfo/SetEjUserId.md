# SetEjUserId(Integer ejUserId)

ID of the ej user record corresponding to this associate; 0 for associates that are not ej users

**Parameter:** 
 - **ejUserId** Integer

```crmscript
NSUserInfo thing;
Integer ejUserId;
thing.SetEjUserId(ejUserId);
```

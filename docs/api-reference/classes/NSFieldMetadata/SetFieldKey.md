# SetFieldKey(String fieldKey)

Internal key/name of the field. Used as the key in the ConnectionConfigField values when TestConnection is called.

**Parameter:** 
 - **fieldKey** String

```crmscript
NSFieldMetadata thing;
String fieldKey;
thing.SetFieldKey(fieldKey);
```

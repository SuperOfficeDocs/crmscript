# SetSecret(String secret)

Shared secret key used for generating SHA256 HMAC signature, so that receiver can verify that call came from this server

**Parameter:** 
 - **secret** String

```crmscript
NSWebhook thing;
String secret;
thing.SetSecret(secret);
```

# SetSecretValue(String secretValue)

Data stored encrypted in the db. Typically a password. Max 70 characters.

**Parameter:** 
 - **secretValue** String

```crmscript
NSUntrustedCredentials thing;
String secretValue;
thing.SetSecretValue(secretValue);
```

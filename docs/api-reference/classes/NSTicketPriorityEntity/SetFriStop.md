# SetFriStop(DateTime friStop)

The work hour stop for Fridays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **friStop** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime friStop;
thing.SetFriStop(friStop);
```

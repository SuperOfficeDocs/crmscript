# SetExtraField3(String extraField3)

This a simple field for adding information that the Connector can provide, and that the qoute document need to display.

**Parameter:** 
 - **extraField3** String

```crmscript
NSQuoteLine thing;
String extraField3;
thing.SetExtraField3(extraField3);
```

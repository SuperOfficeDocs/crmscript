# String[] GetOwnCountryDayTexts()

Array - possibly empty - of the texts related to one day in the redletter system.

**Returns:** String[]

```crmscript
NSRedLetterDetails thing;
String[] ownCountryDayTexts  = thing.GetOwnCountryDayTexts();
```

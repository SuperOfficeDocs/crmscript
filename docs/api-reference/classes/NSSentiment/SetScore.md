# SetScore(Integer score)

Sentiment score, -100 = very unhappy, 100 = very happy, 0 = no idea (not recognized)

**Parameter:** 
 - **score** Integer

```crmscript
NSSentiment thing;
Integer score;
thing.SetScore(score);
```

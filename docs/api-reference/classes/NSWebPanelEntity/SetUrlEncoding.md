# SetUrlEncoding(Integer urlEncoding)

The encoding of the URL

**Parameter:** 
 - **urlEncoding** Integer
     - Enum: 0 = Unknown 
     - Enum: 1 = None 
     - Enum: 2 = ANSI 
     - Enum: 3 = Unicode 

```crmscript
NSWebPanelEntity thing;
Integer urlEncoding;
thing.SetUrlEncoding(urlEncoding);
```

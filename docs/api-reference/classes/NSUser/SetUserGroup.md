# SetUserGroup(NSUserGroup userGroup)

The main user group that this user belongs to.

**Parameter:** 
 - **userGroup** NSUserGroup

```crmscript
NSUser thing;
NSUserGroup userGroup;
thing.SetUserGroup(userGroup);
```

# SetSearchCompany(Bool searchCompany)

True if you want to search for Companies by their name.

**Parameter:** 
 - **searchCompany** Bool

```crmscript
NSPhoneListPreferences thing;
Bool searchCompany;
thing.SetSearchCompany(searchCompany);
```

# SetAlertTimeout(DateTime alertTimeout)

The datetime for when the ticket should jump to the next alert_level.

**Parameter:** 
 - **alertTimeout** DateTime

```crmscript
NSTicket thing;
DateTime alertTimeout;
thing.SetAlertTimeout(alertTimeout);
```

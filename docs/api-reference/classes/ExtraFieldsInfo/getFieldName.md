# ExtraFieldsInfo.getFieldName()

This function returns the field name of the current field.

## Example

This code prints all field names of extra fields:

    ExtraFieldsInfo e;
    
    e.getExtraFields("person");
    
    while(!e.eof()){
      print(e.getFieldName());
      e.next();
    }

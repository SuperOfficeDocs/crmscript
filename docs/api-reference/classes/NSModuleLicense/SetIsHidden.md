# SetIsHidden(Bool isHidden)

If true, then this module should not appear in GUI and is not assignable by users

**Parameter:** 
 - **isHidden** Bool

```crmscript
NSModuleLicense thing;
Bool isHidden;
thing.SetIsHidden(isHidden);
```

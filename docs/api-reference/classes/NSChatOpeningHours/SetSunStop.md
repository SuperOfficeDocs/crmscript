# SetSunStop(TimeSpan sunStop)

When opening hours stop on Sundays. Only time-part is used.

**Parameter:** 
 - **sunStop** TimeSpan

```crmscript
NSChatOpeningHours thing;
TimeSpan sunStop;
thing.SetSunStop(sunStop);
```

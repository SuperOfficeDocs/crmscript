# SetSlevel(TicketSecurityLevel slevel)

The securitylevel of the ticket.

**Parameter:** 
 - **slevel** TicketSecurityLevel
     - Enum: 0 = Unknown 
     - Enum: 1 = Internal 
     - Enum: 2 = External 

```crmscript
NSTicketEntity thing;
TicketSecurityLevel slevel;
thing.SetSlevel(slevel);
```

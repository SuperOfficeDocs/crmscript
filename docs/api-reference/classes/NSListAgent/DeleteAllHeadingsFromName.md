# Void DeleteAllHeadingsFromName(String name)

Delete all headings for list resolved by the provided name.

**Parameters:**
 - **name** The name of the list to look up.

```crmscript
NSListAgent agent;
String name;
agent.DeleteAllHeadingsFromName(name);
```

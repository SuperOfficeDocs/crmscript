# SetState(Integer state)

Checkout state

**Parameter:** 
 - **state** Integer
     - Enum: 0 = NotCheckedOut 
     - Enum: 1 = CheckedOutOwn 
     - Enum: 2 = CheckedOutOther 
     - Enum: 3 = LockingNotSupported 

```crmscript
NSCheckoutInfo thing;
Integer state;
thing.SetState(state);
```

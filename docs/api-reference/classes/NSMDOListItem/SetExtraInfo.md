# SetExtraInfo(String extraInfo)

Extra information added to the ListItem. Could be information such as sort order etc or other meta data. Custom field.

**Parameter:** 
 - **extraInfo** String

```crmscript
NSMDOListItem thing;
String extraInfo;
thing.SetExtraInfo(extraInfo);
```

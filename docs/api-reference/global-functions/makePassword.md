# String makePassword()

Creates and returns a random password of the given length. You can optionally pass the given seed characters.

## Functions

* String makePassword(Integer length)
* String makePassword(Integer length, String seed)

## Parameters

| Parameter | Type | Description |
|---|---|---|
| length | Integer | Length of password |
| seed | String | Optional. Valid characters |

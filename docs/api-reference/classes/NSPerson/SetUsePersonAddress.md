# SetUsePersonAddress(Bool usePersonAddress)

If 1, use person's address for mailing instead of company address

**Parameter:** 
 - **usePersonAddress** Bool

```crmscript
NSPerson thing;
Bool usePersonAddress;
thing.SetUsePersonAddress(usePersonAddress);
```

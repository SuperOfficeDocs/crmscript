# SetRawExtraInfo(String rawExtraInfo)

Read-only extra information in plain text, supplied by QuoteConnector, used by a future dynamic GUI extension

**Parameter:** 
 - **rawExtraInfo** String

```crmscript
NSQuoteLine thing;
String rawExtraInfo;
thing.SetRawExtraInfo(rawExtraInfo);
```

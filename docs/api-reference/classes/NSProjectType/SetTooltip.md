# SetTooltip(String tooltip)

Set the tooltip description for the NSProjectType list item.

**Parameter:** 
 - **tooltip** String NSProjectType tooltip description.

```crmscript
NSProjectType thing;
String descr = "something new";
thing.SetTooltip(descr);
```

# Integer GetVisibility()

**Returns:** Integer

     - Enum: 0 = All 
     - Enum: 1 = Associate 
     - Enum: 2 = Group 

```crmscript
NSVisibleFor thing;
Integer visibility  = thing.GetVisibility();
```

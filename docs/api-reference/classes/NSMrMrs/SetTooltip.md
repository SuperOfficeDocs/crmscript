# SetTooltip(String tooltip)

Set the tooltip description for the NSMrMrs list item.

**Parameter:** 
 - **tooltip** String NSMrMrs tooltip description.

```crmscript
NSMrMrs thing;
String descr = "something new";
thing.SetTooltip(descr);
```

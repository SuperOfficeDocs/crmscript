# SetCanRespond(Bool canRespond)

True if the user can respond to chats in this topic

**Parameter:** 
 - **canRespond** Bool

```crmscript
NSChatTopicAgent thing;
Bool canRespond;
thing.SetCanRespond(canRespond);
```

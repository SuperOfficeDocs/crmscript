# SetCredentials(NSCredential[] credentials)

List of credentials registered for this user. i.e. valid authentication methods.

**Parameter:** 
 - **credentials** NSCredential[]

```crmscript
NSUser thing;
NSCredential[] credentials;
thing.SetCredentials(credentials);
```

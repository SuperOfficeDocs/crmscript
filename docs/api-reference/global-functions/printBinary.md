# Void printBinary()

Output the bytes to the standard output stream. You can pass the data to encode as either Byte[] or NSStream.

## Functions

* Void printBinary(Byte[] bytes)
* Void printBinary(NSStream bytes)

## Parameters

| Parameter | Type | Description |
|---|---|---|
| bytes | Byte[] OR<br>NSStream | The bytes to output |

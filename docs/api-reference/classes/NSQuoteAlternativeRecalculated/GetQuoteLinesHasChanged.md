# Bool GetQuoteLinesHasChanged()

If true any of the quotelines for the alternative has been changed.

**Returns:** Bool

```crmscript
NSQuoteAlternativeRecalculated thing;
Bool quoteLinesHasChanged  = thing.GetQuoteLinesHasChanged();
```

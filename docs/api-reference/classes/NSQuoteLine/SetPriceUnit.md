# SetPriceUnit(String priceUnit)

What is the unit (meter, ton, bushel, microsecond, gradus, τρυβλίον, 五合枡, دونم or whatever); read-only for lines that originate in defined products.

**Parameter:** 
 - **priceUnit** String

```crmscript
NSQuoteLine thing;
String priceUnit;
thing.SetPriceUnit(priceUnit);
```

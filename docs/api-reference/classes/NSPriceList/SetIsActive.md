# SetIsActive(Bool isActive)

Is the list active (as opposed to being worked on, suddenly canceled, etc.

**Parameter:** 
 - **isActive** Bool

```crmscript
NSPriceList thing;
Bool isActive;
thing.SetIsActive(isActive);
```

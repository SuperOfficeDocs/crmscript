Get all user groups

**Parameters:**
 - **includeDeleted** Include user groups with Deleted set to true

**Returns:** All user groups

```crmscript
NSUserAgent agent;
Bool includeDeleted;
UserGroup[] res = agent.GetAllUserGroups(includeDeleted);
```

# ReplyTemplate.getPlainBody(Integer language)

Returns the Reply template plain text body, in the language given by language

## Parameters

 - language as an integer, see below

## Languages

Norwegian = 0,
English = 1,
German = 2,
Swedish = 3,
Danish = 4,
Dutch = 5,

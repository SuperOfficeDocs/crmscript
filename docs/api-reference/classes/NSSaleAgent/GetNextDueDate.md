Gets the next due date for a sale. The next due date is the next future appointment that is not completed.

**Parameters:**
 - **saleId** Sale id

**Returns:** DateTime

```crmscript
NSSaleAgent agent;
Integer saleId;
DateTime res = agent.GetNextDueDate(saleId);
```

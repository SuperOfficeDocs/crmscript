# SetSunStart(DateTime sunStart)

The work hour start for Sundays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **sunStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime sunStart;
thing.SetSunStart(sunStart);
```

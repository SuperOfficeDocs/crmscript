# SetThuStart(DateTime thuStart)

The work hour start for Thursdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **thuStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime thuStart;
thing.SetThuStart(thuStart);
```

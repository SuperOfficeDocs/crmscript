# SetFolderId(Integer folderId)

The folder which this message belongs to. -1 indicates that the message is on the root

**Parameter:** 
 - **folderId** Integer

```crmscript
NSShipmentMessageEntity thing;
Integer folderId;
thing.SetFolderId(folderId);
```

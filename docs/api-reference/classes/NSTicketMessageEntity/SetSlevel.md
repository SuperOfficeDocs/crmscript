# SetSlevel(TicketSecurityLevel slevel)

The securitylevel of the message.

**Parameter:** 
 - **slevel** TicketSecurityLevel
     - Enum: 0 = Unknown 
     - Enum: 1 = Internal 
     - Enum: 2 = External 

```crmscript
NSTicketMessageEntity thing;
TicketSecurityLevel slevel;
thing.SetSlevel(slevel);
```

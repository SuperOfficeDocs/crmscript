# SetMailingsProviderName(String mailingsProviderName)

**Parameter:** 
 - **mailingsProviderName** String

```crmscript
NSSelectionEntity thing;
String mailingsProviderName;
thing.SetMailingsProviderName(mailingsProviderName);
```

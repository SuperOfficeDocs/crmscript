# SetWedStart(DateTime wedStart)

The work hour start for Wednesdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **wedStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime wedStart;
thing.SetWedStart(wedStart);
```

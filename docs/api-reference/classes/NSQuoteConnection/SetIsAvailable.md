# SetIsAvailable(Bool isAvailable)

Whether or not the specified connection is available. Typically, without network access the availability is false.

**Parameter:** 
 - **isAvailable** Bool

```crmscript
NSQuoteConnection thing;
Bool isAvailable;
thing.SetIsAvailable(isAvailable);
```

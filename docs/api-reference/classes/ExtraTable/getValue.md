# ExtraTable.getValue(String field)

This function returns the value of the given field for the currently loaded entry.

## Parameters

 - field: The name of the extra table field - e.g. "x\_myfield"

## Example

    ExtraTable ex;
    
    Bool b = ex.load(2);
    print(b.toString());
    
    print(ex.getValue("x_table"));

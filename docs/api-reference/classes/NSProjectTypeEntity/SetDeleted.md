# SetDeleted(Bool deleted)

0 -> record is active 1 -> record is 'deleted' and should not be shown in lists

**Parameter:** 
 - **deleted** Bool

```crmscript
NSProjectTypeEntity thing;
Bool deleted;
thing.SetDeleted(deleted);
```

# SetTooltip(String tooltip)

Set the tooltip description for the NSPaymentTerm list item.

**Parameter:** 
 - **tooltip** String NSPaymentTerm tooltip description.

```crmscript
NSPaymentTerm thing;
String descr = "something new";
thing.SetTooltip(descr);
```

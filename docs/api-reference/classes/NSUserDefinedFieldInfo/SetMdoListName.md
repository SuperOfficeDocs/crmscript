# SetMdoListName(String mdoListName)

MDO list name used to populate this list. Derived from UDListDefinitionId and ListTableId. (Read-only)

**Parameter:** 
 - **mdoListName** String

```crmscript
NSUserDefinedFieldInfo thing;
String mdoListName;
thing.SetMdoListName(mdoListName);
```

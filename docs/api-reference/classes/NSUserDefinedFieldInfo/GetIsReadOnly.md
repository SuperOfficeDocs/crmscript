# Bool GetIsReadOnly()

0 = read/write, 1 = readonly (don't combine with mandatory  8-) )

**Returns:** Bool

```crmscript
NSUserDefinedFieldInfo thing;
Bool isReadOnly  = thing.GetIsReadOnly();
```

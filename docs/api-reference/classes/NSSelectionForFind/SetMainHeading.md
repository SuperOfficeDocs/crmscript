# SetMainHeading(String mainHeading)

'Sale', intended for the main headings in the Find front page; this string will contain resource references

**Parameter:** 
 - **mainHeading** String

```crmscript
NSSelectionForFind thing;
String mainHeading;
thing.SetMainHeading(mainHeading);
```

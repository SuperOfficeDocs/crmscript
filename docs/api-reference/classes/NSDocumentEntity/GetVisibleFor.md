# NSVisibleFor[] GetVisibleFor()

Return the usergroups and/or users the NSDocumentEntity item is visible for.

**Returns:** NSVisibleFor[]

```crmscript
NSDocumentEntity thing;
NSVisibleFor[] visibleFor = thing.GetVisibleFor();
```

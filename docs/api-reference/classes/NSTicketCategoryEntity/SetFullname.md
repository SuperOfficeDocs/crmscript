# SetFullname(String fullname)

The fullname (compiled from parents&apos; names) for this category. E.g. &apos;norway/product A/support&apos;.

**Parameter:** 
 - **fullname** String

```crmscript
NSTicketCategoryEntity thing;
String fullname;
thing.SetFullname(fullname);
```

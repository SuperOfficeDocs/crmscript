# SetMandatory(Bool mandatory)

Should the user be forced to set a value (the Erp Connector may declare fields mandatory as well)

**Parameter:** 
 - **mandatory** Bool

```crmscript
NSErpSyncDefaultValue thing;
Bool mandatory;
thing.SetMandatory(mandatory);
```

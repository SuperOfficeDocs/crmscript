# String GetShortLabel()

Short name to be used in Archive headings and on page 1. If blank, the fieldLabel will be used everywhere.

**Returns:** String

```crmscript
NSUserDefinedFieldInfo thing;
String shortLabel  = thing.GetShortLabel();
```

# SetAssociateId(Integer associateId)

 Task owner. If it is a System task, AssociateId = 0.

**Parameter:** 
 - **associateId** Integer

```crmscript
NSBatchTaskInfo thing;
Integer associateId;
thing.SetAssociateId(associateId);
```

# SetCompleted(Bool completed)

Done (0=false, 1=true). Status implies changes in which fields are shown in GUI, as well as which fields can be updated

**Parameter:** 
 - **completed** Bool

```crmscript
NSProjectEntity thing;
Bool completed;
thing.SetCompleted(completed);
```

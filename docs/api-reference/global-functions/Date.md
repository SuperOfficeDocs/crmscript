# Date Date()

Date constructor. You can pass either a `Date` object to copy or a `String` to be parsed.

Returns a `Date` object.

## Date Date(Date date)

`Date` object to copy.

## Date Date(String date)

Parse `String` with format **YYYY-MM-DD** and create `Date` object.

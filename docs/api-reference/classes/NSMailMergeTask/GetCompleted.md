# Integer GetCompleted()

Completed status for task

**Returns:** Integer

     - Enum: 0 = Unknown 
     - Enum: 1 = NotStarted 
     - Enum: 2 = Started 
     - Enum: 3 = Completed 

```crmscript
NSMailMergeTask thing;
Integer completed  = thing.GetCompleted();
```

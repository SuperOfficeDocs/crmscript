# SetReopenDate(DateTime reopenDate)

Date the sale is to be reopened; valid only for status=stalled. Not necessarily the same as the nextDueDate.

**Parameter:** 
 - **reopenDate** DateTime

```crmscript
NSSaleEntity thing;
DateTime reopenDate;
thing.SetReopenDate(reopenDate);
```

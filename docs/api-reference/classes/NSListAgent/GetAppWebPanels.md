Get all web panels owned by your app; works in Online only, for registered Apps that send a valid ApplicationToken


**Returns:** NSWebPanelEntity[]

```crmscript
NSListAgent agent;
NSWebPanelEntity[] res = agent.GetAppWebPanels();
```

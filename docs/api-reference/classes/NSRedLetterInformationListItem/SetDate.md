# SetDate(DateTime date)

Date that this item is valid for; there is exactly one item per date, ordered by date.

**Parameter:** 
 - **date** DateTime

```crmscript
NSRedLetterInformationListItem thing;
DateTime date;
thing.SetDate(date);
```

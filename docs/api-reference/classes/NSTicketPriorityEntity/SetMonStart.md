# SetMonStart(DateTime monStart)

The work hour start for Mondays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **monStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime monStart;
thing.SetMonStart(monStart);
```

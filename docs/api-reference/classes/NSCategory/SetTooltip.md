# SetTooltip(String tooltip)

Set the tooltip description for the NSCategory list item.

**Parameter:** 
 - **tooltip** String NSCategory tooltip description.

```crmscript
NSCategory thing;
String descr = "something new";
thing.SetTooltip(descr);
```

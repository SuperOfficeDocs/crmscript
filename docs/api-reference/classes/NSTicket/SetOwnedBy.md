# SetOwnedBy(Integer ownedBy)

The id of the user who owns the ticket. 1 (system user) if the ticket is unassigned.

**Parameter:** 
 - **ownedBy** Integer

```crmscript
NSTicket thing;
Integer ownedBy;
thing.SetOwnedBy(ownedBy);
```

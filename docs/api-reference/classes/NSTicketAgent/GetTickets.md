**Parameters:**
 - **ticketIds** The ids of tickets to get

**Returns:** Returns array of tickets in same order as input ids

```crmscript
NSTicketAgent agent;
Integer[] ticketIds;
Ticket[] res = agent.GetTickets(ticketIds);
```

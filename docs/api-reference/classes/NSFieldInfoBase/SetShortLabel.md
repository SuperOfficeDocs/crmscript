# SetShortLabel(String shortLabel)

Short name to be used in Archive headings and on page 1. If blank, the fieldLabel will be used everywhere.

**Parameter:** 
 - **shortLabel** String

```crmscript
NSFieldInfoBase thing;
String shortLabel;
thing.SetShortLabel(shortLabel);
```

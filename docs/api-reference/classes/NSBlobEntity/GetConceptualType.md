# String GetConceptualType()

The type, for instance PHOTO, PERSONPHOTO, or whatever, that is descriptive of what kind of image or data this is

**Returns:** String

```crmscript
NSBlobEntity thing;
String conceptualType  = thing.GetConceptualType();
```

# SetBlockExecution(Bool blockExecution)

Whether the context should stop what it is doing, e.g. saving a sale

**Parameter:** 
 - **blockExecution** Bool

```crmscript
NSEventData thing;
Bool blockExecution;
thing.SetBlockExecution(blockExecution);
```

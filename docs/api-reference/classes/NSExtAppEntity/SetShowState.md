# SetShowState(Integer showState)

The application is shown in which state Minimized, Maximized or Restored

**Parameter:** 
 - **showState** Integer
     - Enum: 0 = Minimized 
     - Enum: 1 = Maximized 
     - Enum: 2 = Default 
     - Enum: 256 = ToolBar 
     - Enum: 512 = AddressBar 
     - Enum: 1024 = StatusBar 
     - Enum: 2048 = MenuBar 

```crmscript
NSExtAppEntity thing;
Integer showState;
thing.SetShowState(showState);
```

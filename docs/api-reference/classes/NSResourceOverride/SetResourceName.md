# SetResourceName(String resourceName)

The name of the resource, without any brackets or braces - example: SR_CONTACT

**Parameter:** 
 - **resourceName** String

```crmscript
NSResourceOverride thing;
String resourceName;
thing.SetResourceName(resourceName);
```

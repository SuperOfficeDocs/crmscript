# SetSubGroup(String subGroup)

Keys that share a value (including NULL) in this field are shown together; a visual spacer is shown between groups

**Parameter:** 
 - **subGroup** String

```crmscript
NSPreferenceDescription thing;
String subGroup;
thing.SetSubGroup(subGroup);
```

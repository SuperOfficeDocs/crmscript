# SetSunStop(DateTime sunStop)

The work hour stop for Sundays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **sunStop** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime sunStop;
thing.SetSunStop(sunStop);
```

# SetPublicKey(NSSignedPublicKey publicKey)

The public key used to verify licenses owned by this owner.

**Parameter:** 
 - **publicKey** NSSignedPublicKey

```crmscript
NSLicenseInfo thing;
NSSignedPublicKey publicKey;
thing.SetPublicKey(publicKey);
```

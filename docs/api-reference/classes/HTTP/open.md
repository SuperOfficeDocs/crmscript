# HTTP.open(String p_url)

This function opens an url and returns the result as a Byte array.

## Parameters

 - p\_url: The url to open. http://.....

## Example

    HTTP h;
    
    Byte[] b = h.open("https://httpbin.org/");
    
    if (h.hasError())
      print(h.getErrorMessage());
    else
      print(String(b));

This entity supports Soft Delete. Call this method to Undelete a previously soft-deleted record

**Parameters:**
 - **id** The primary key of the entity to undelete

**Returns:**

```crmscript
NSContactAgent agent;
Integer id;
agent.Undelete(id);
```

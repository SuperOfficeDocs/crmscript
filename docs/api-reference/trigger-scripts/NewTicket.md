# NewTicket (100)

Runs when a new ticket is created.

## Input values
|Variable|Description|
|---|---|
| `entryId` | request ID|
| `ticketId` | request ID|
| `personId` | customer (person) ID|
| `contactId` | company (contact) ID|

## Sample code

```crmscript
#setLanguageLevel 3;
String param1 = getVariable("customerId");
```

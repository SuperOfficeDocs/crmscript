Returns a single list item

**Parameters:**
 - **listName** Conceptual name of the MDO list
 - **id** Id of list item

**Returns:** Single MDO list item

```crmscript
NSMDOAgent agent;
String listName;
Integer id;
NSMDOListItem res = agent.GetListItem(listName, id);
```

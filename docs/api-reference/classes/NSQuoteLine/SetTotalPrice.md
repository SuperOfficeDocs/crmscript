# SetTotalPrice(Float totalPrice)

TotalPrice  = SubTotal - DiscountAmount or TotalPrice = (UnitCost * Quantity) + EarningAmount, according to what the user changed last.

**Parameter:** 
 - **totalPrice** Float

```crmscript
NSQuoteLine thing;
Float totalPrice;
thing.SetTotalPrice(totalPrice);
```

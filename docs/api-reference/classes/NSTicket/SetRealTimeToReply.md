# SetRealTimeToReply(Integer realTimeToReply)

Same as time_to_reply, but not calculated based on priority.

**Parameter:** 
 - **realTimeToReply** Integer

```crmscript
NSTicket thing;
Integer realTimeToReply;
thing.SetRealTimeToReply(realTimeToReply);
```

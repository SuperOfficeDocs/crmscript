# SetSize(Integer size)

Chat widget size: normal or large.

**Parameter:** 
 - **size** Integer
     - Enum: 0 = Normal 
     - Enum: 1 = Large 

```crmscript
NSChatWidgetSettings thing;
Integer size;
thing.SetSize(size);
```

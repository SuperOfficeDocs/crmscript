# String GetJoinVideomeetUrl()

Blank when not a video meeting. Filled with Join Meeting URL when created.

**Returns:** String

```crmscript
NSAppointment thing;
String joinVideomeetUrl  = thing.GetJoinVideomeetUrl();
```

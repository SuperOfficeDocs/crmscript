# SetRecordId(Integer recordId)

Record id in target table, used when there are separate counters for each target record

**Parameter:** 
 - **recordId** Integer

```crmscript
NSRefCountEntity thing;
Integer recordId;
thing.SetRecordId(recordId);
```

# SetAlertLevel(Integer alertLevel)

The alert level for the ticket. Matches the level value of the ticket_alert table.

**Parameter:** 
 - **alertLevel** Integer

```crmscript
NSTicket thing;
Integer alertLevel;
thing.SetAlertLevel(alertLevel);
```

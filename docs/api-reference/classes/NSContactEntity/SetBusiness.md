# SetBusiness(NSBusiness business)

The business that the contact is associated with. The GUI forces the user to enter a business type.

**Parameter:** 
 - **business** NSBusiness

```crmscript
NSContactEntity thing;
NSBusiness business;
thing.SetBusiness(business);
```

# SetCustLangId(Integer custLangId)

The customer language which this config belongs to. 0 indicates that the config is global

**Parameter:** 
 - **custLangId** Integer

```crmscript
NSCustomerCenterConfig thing;
Integer custLangId;
thing.SetCustLangId(custLangId);
```

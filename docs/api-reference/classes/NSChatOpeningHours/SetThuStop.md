# SetThuStop(TimeSpan thuStop)

When opening hours stop on Thursdays. Only time-part is used.

**Parameter:** 
 - **thuStop** TimeSpan

```crmscript
NSChatOpeningHours thing;
TimeSpan thuStop;
thing.SetThuStop(thuStop);
```

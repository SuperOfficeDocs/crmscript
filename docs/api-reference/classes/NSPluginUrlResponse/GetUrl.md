# String GetUrl()

Url that the GUI should navigato to/open, if non-blank. The GUI cannot enforce any rules subsequent to opening the requested url.

**Returns:** String

```crmscript
NSPluginUrlResponse thing;
String url  = thing.GetUrl();
```

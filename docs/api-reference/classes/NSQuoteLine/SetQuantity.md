# SetQuantity(Float quantity)

How many units; this is a decimal field since you might want to offer fractional units (2.5kg, or 0.5PC).

**Parameter:** 
 - **quantity** Float

```crmscript
NSQuoteLine thing;
Float quantity;
thing.SetQuantity(quantity);
```

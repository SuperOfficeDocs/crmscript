# SetCategoryName(String categoryName)

The (short) name of this category. E.g. &apos;support&apos;.

**Parameter:** 
 - **categoryName** String

```crmscript
NSTicket thing;
String categoryName;
thing.SetCategoryName(categoryName);
```

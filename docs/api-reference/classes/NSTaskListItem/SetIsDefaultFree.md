# SetIsDefaultFree(Bool isDefaultFree)

True if free, false if busy

**Parameter:** 
 - **isDefaultFree** Bool

```crmscript
NSTaskListItem thing;
Bool isDefaultFree;
thing.SetIsDefaultFree(isDefaultFree);
```

# SetProperties(StringObjectDictionary properties)

Custom values to inject into JSON body of webhook call.

**Parameter:** 
 - **properties** StringObjectDictionary

```crmscript
NSWebhook thing;
StringObjectDictionary properties;
thing.SetProperties(properties);
```

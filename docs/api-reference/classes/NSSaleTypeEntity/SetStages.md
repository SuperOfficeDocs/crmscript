# SetStages(NSSelectableMDOListItem[] stages)

**Parameter:** 
 - **stages** NSSelectableMDOListItem[]

```crmscript
NSSaleTypeEntity thing;
NSSelectableMDOListItem[] stages;
thing.SetStages(stages);
```

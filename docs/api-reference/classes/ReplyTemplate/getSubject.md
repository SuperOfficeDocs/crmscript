# ReplyTemplate.getSubject(Integer language)

Returns the Reply template subject, in the language given by language. The reply template subject is in plain text only.

## Parameters

 - language as an integer, see below

## Languages

Norwegian = 0,
English = 1,
German = 2,
Swedish = 3,
Danish = 4,
Dutch = 5,

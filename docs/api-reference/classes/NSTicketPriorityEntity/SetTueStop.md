# SetTueStop(DateTime tueStop)

The work hour stop for Tuesdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **tueStop** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime tueStop;
thing.SetTueStop(tueStop);
```

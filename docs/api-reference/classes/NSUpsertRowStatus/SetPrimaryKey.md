# SetPrimaryKey(Integer primaryKey)

The primary key of the target table, that matches the keys specified in the data row

**Parameter:** 
 - **primaryKey** Integer

```crmscript
NSUpsertRowStatus thing;
Integer primaryKey;
thing.SetPrimaryKey(primaryKey);
```

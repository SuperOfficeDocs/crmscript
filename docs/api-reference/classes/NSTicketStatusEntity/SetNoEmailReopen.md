# SetNoEmailReopen(Bool noEmailReopen)

Whether inbound emails can reopen requests with this status or not

**Parameter:** 
 - **noEmailReopen** Bool

```crmscript
NSTicketStatusEntity thing;
Bool noEmailReopen;
thing.SetNoEmailReopen(noEmailReopen);
```

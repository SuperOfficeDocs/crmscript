# SetFormalName(String formalName)

Get formal name for a person, as used in labels. (Full name + person title + academic title)

**Parameter:** 
 - **formalName** String

```crmscript
NSPersonEntity thing;
String formalName;
thing.SetFormalName(formalName);
```

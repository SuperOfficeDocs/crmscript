# SetUrl(String url)

A url to the product info. Can be empty. Is stored here if the user changes the value from the product in the pricelist, or just enters a QuoteLine without a product link.

**Parameter:** 
 - **url** String

```crmscript
NSQuoteLine thing;
String url;
thing.SetUrl(url);
```

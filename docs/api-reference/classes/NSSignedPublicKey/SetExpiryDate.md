# SetExpiryDate(DateTime expiryDate)

Date when the signature expires. No Timezone.

**Parameter:** 
 - **expiryDate** DateTime

```crmscript
NSSignedPublicKey thing;
DateTime expiryDate;
thing.SetExpiryDate(expiryDate);
```

**Parameters:**
 - **uniqueIdentifier** Global unique reference to script (independent of installation)

**Returns:** String

```crmscript

String uniqueIdentifier;
Script res = agent.GetCRMScriptByUniqueIdentifier(uniqueIdentifier);
```

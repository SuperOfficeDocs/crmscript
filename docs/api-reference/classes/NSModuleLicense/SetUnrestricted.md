# SetUnrestricted(Bool unrestricted)

If true, then there is no licenseNumber limit on how many of these modules may be assigned

**Parameter:** 
 - **unrestricted** Bool

```crmscript
NSModuleLicense thing;
Bool unrestricted;
thing.SetUnrestricted(unrestricted);
```

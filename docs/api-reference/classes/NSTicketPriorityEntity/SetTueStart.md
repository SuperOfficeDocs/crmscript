# SetTueStart(DateTime tueStart)

The work hour start for Tuesdays. Note that only the time part of the DateTime is used

**Parameter:** 
 - **tueStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime tueStart;
thing.SetTueStart(tueStart);
```

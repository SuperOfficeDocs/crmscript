# SetActionTime(DateTime actionTime)

The time of the activity. If more than one activity has occured on the contact, the last activity time is shown.

**Parameter:** 
 - **actionTime** DateTime

```crmscript
NSContactActivity thing;
DateTime actionTime;
thing.SetActionTime(actionTime);
```

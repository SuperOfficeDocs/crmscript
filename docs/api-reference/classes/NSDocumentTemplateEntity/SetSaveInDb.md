# SetSaveInDb(Integer saveInDb)

1 = save document records in DB; otherwise not.

**Parameter:** 
 - **saveInDb** Integer

```crmscript
NSDocumentTemplateEntity thing;
Integer saveInDb;
thing.SetSaveInDb(saveInDb);
```

# Void Revert(Integer ownerType)

Revert changed Udef fields for the given owner type - all unpublished changes will be lost

**Parameters:**
 - **ownerType** The owner type to revert fields for
     - Enum: 0 = Invalid 
     - Enum: 7 = Contact 
     - Enum: 8 = Person 
     - Enum: 9 = Project 
     - Enum: 10 = Sale 
     - Enum: 11 = Temp 
     - Enum: 12 = Appointment 
     - Enum: 13 = Document 
     - Enum: 14 = None

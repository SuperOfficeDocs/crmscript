# SetItemInfo(String itemInfo)

Extra information on the history item, e.g. The Associate Type for an associate or other relevant info.

**Parameter:** 
 - **itemInfo** String

```crmscript
NSHistory thing;
String itemInfo;
thing.SetItemInfo(itemInfo);
```

# SetRowKeys(String[] rowKeys)

The key values given in the incoming 'data' parameter to the Upsert call, in the order they were speicifed in the 'keys' array

**Parameter:** 
 - **rowKeys** String[]

```crmscript
NSUpsertRowStatus thing;
String[] rowKeys;
thing.SetRowKeys(rowKeys);
```

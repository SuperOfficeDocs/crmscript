# Float GetSubTotal()

Value to help the user interface, is computed by summing the totalprice of the quotelines, and NOT by summing their subtotals

**Returns:** Float

```crmscript
NSQuoteAlternative thing;
Float subTotal  = thing.GetSubTotal();
```

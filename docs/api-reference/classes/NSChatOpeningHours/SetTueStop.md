# SetTueStop(TimeSpan tueStop)

When opening hours stop on Tuesdays. Only time-part is used.

**Parameter:** 
 - **tueStop** TimeSpan

```crmscript
NSChatOpeningHours thing;
TimeSpan tueStop;
thing.SetTueStop(tueStop);
```

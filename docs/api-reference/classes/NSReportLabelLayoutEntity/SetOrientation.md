# SetOrientation(Integer orientation)

Paper orientation: 1=portrait, 2=landscape

**Parameter:** 
 - **orientation** Integer
     - Enum: 0 = None 
     - Enum: 1 = Portrait 
     - Enum: 2 = Landscape 

```crmscript
NSReportLabelLayoutEntity thing;
Integer orientation;
thing.SetOrientation(orientation);
```

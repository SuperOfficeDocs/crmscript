# Void SetEnabled(Bool enabled)

Sets freetext search to enabled (true) or disabled (false)

**Parameters:**
 - **enabled** If enabled true, else false

```crmscript
NSFreeTextAgent agent;
Bool enabled;
agent.SetEnabled(enabled);
```

# HTTP.deleteAsStream(String p_0)

Supports both http and https. Returns the result in a NSStream.

## Example

    HTTP h;
    
    NSStream b = h.deleteAsStream("https://httpbin.org/delete");
    if (h.hasError())
      print(h.getErrorMessage());
    else
      print(String(b));

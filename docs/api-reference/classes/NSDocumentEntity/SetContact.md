# SetContact(NSContact contact)

The contact associated with the document. It may also be null if no contact is associated with the document.

**Parameter:** 
 - **contact** NSContact

```crmscript
NSDocumentEntity thing;
NSContact contact;
thing.SetContact(contact);
```

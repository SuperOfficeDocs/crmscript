# SetEventDate(DateTime eventDate)

Date of the event. Event is not visible after this date.

**Parameter:** 
 - **eventDate** DateTime

```crmscript
NSProjectEvent thing;
DateTime eventDate;
thing.SetEventDate(eventDate);
```

# SetSortOrder(Integer sortOrder)

Indicates the sort order for this priority. 1 is first, 100 is last

**Parameter:** 
 - **sortOrder** Integer

```crmscript
NSTicketPriorityEntity thing;
Integer sortOrder;
thing.SetSortOrder(sortOrder);
```

Set the tooltip description for the NSCustomerLanguage list item.

**Parameter:** 
 - **tooltip** String NSCustomerLanguage tooltip description.

```crmscript
NSCustomerLanguage thing;
String descr = "something new";
thing.SetTooltip(descr);
```

# SetDeleted(Integer deleted)

0 -> record is active 1 -> record is 'deleted' and should not be shown in lists

**Parameter:** 
 - **deleted** Integer

```crmscript
NSDocumentTemplate thing;
Integer deleted;
thing.SetDeleted(deleted);
```

# Void SaveTabOrder(NSTabOrder tabOrder)

Saves the tab order. The order is saved pr. user.

**Parameters:**
 - **tabOrder** Name of the tab control

```crmscript
NSPreferenceAgent agent;
NSTabOrder tabOrder;
agent.SaveTabOrder(tabOrder);
```

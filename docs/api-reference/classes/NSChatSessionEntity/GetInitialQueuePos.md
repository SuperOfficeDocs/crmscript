# Integer GetInitialQueuePos()

The initial queue pos (i.e. the number of pending sessions + 1) when the session was requested.

**Returns:** Integer

```crmscript
NSChatSessionEntity thing;
Integer initialQueuePos  = thing.GetInitialQueuePos();
```
